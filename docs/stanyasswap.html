OSI

OSI (open systems interconnection) is conceptual framework used to understand network interactions. developed by ISO (international organization for standardization.)

(1) physical

physical connection between devices. information transferred via bits. hubs, repeaters, modems, cables

(2) data link

transmits packets to host using mac address. doesn't cross boundary of local network. "frame" layer. receivers mac address obtained by placing ARP (address resolution protocol) request through the network (asking how has this [IP] ??) and destination host replies with its MAC. . devices used are switches and bridges.

(3) network

transmission of data across different networks. packet routing (selection of shortest path to transmit the packet). sender/receiver IP placed in header. packet layer. devices used are routers and switches.

(4) transport

responsible for end-to-end delivery of message. uses TCP (transmission control protocol) or UDP (user datagram protocol). TCP provides acknowledgment of successful data transmission and re-transmits data if any errors are found. UDP doesn't have these features.

(5) session

responsible for establishment/management/termination of connections/sessions between two devices. provides authentication and security.

(6) presentation

provides translation. encryption/decryption. compression.

(7) application

displaying the received information to the user
